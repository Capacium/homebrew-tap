# Capacium Homebrew Tap

Homebrew formula for [Capacium](https://github.com/Capacium/capacium) — Capability Packaging System for AI agent capabilities.

## Usage

```bash
brew tap capacium/tap
brew install capacium
```

Or directly:

```bash
brew install capacium/tap/capacium
```

## What is Capacium?

Capacium defines a standard manifest format (`capability.yaml`), a CLI (`cap`) for package management, and a trust model based on SHA-256 fingerprinting and Ed25519 signing. It supports six capability kinds — skills, bundles, tools, prompts, templates, and workflows — and works across OpenCode, Claude Code, Gemini CLI, Cursor, and Continue.dev.
